# Orion Agents

Each agent is a specialized module responsible for one phase of the organizational DNA extraction process.

## Agent Index

- `interviewer/` — Conducts structured discovery sessions
- `mapper/` — Converts raw interviews into graph nodes  
- `architect/` — Designs system architecture
- `builder/` — Generates entities and workflows
- `validator/` — Tests against business rules
- `integrator/` — Connects external APIs
- `deployer/` — Production deployment
- `monitor/` — Adoption tracking
