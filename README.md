## What is Orion?

Traditional software takes 6-18 months and costs a fortune. Orion does it in 30 days.

Orion is a multi-agent AI system that:

1. **Interviews** your team's experts through structured conversations
2. **Maps** your tacit knowledge into an organizational DNA graph
3. **Builds** a fully functional custom OS for your business — automatically
4. **Deploys** it in production, ready for your team to use

## How it works

```
Your Team's Knowledge
        ↓
   Orion Agents (interview + extract)
        ↓
   Organizational DNA Graph
        ↓
   Custom Business OS (deployed in 30 days)
```

### The Agent Architecture

Orion runs 8 specialized AI agents in parallel:

| Agent | Role |
|-------|------|
| `Interviewer` | Structured discovery sessions with domain experts |
| `Mapper` | Converts interviews into knowledge graph nodes |
| `Architect` | Designs the system architecture from the graph |
| `Builder` | Generates entities, workflows and automations |
| `Validator` | Tests logic against real business rules |
| `Integrator` | Connects external APIs and data sources |
| `Deployer` | Pushes to production environment |
| `Monitor` | Tracks adoption and system performance |

## Get Started

```bash
# Clone the repository
git clone https://github.com/b2bytes/orion.git
cd orion

# Install dependencies
npm install

# Configure your environment
cp .env.example .env

# Run your first interview session
npm run orion:start
```
